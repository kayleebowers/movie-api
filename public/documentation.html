myFlix

Introduction

myFlix is a Node.js and Express application that provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Return a list of ALL movies to the user — GET — /movies

Return data about a single movie by title to the user — GET — /movies/titles

Return data about a genre (description) by name/title —GET — /movies/genres

Return data about a director (bio, birth year, death year) by name —GET /movies/directors

Allow new users to register —POST /register

Allow users to update their user info (username) —PUT /update_information

Allow users to add a movie to their list of favorites —POST /add

Allow users to remove a movie from their list of favorites —DELETE /delete

Allow existing users to deregister —DELETE /deregister
